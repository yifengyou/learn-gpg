# GnuPG/GPG

![20200107_170300_31](image/20200107_170300_31.png)

* **GNU Privacy Guard（GnuPG或GPG）**是一个密码学软件，用于加密、签名通信内容及管理非对称密码学的密钥。GnuPG是自由软件，遵循IETF订定的OpenPGP技术标准设计，并与PGP保持兼容。
* GnuPG是自由软件基金会的GNU计划的一部分，曾受德国政府资助。
